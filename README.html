Directory Structure
  • BillsToLaws : Project Root Directory
    • build : Built files
      • classes : Built classes
        • deibert : Built classes in the deibert package
          • zach : Built classes in the deibert.zach package
            • school : Built classes in the deibert.zach.school package
              • billstolaws : Built classes in the deibert.zach.school.billstolaws package
                • tiles : Built classes in the deibert.zach.school.billstolaws.tiles package
                • uim : Built classes in the deibert.zach.school.billstolaws.uim package
      • empty : An empty directory...
      • generated-sources : All generated sources
    • dist : Distribution files
    • nbproject : NetBeans IDE metadata
    • src : Sources
      • deibert : Sources of classes in the deibert package
        • zach : Sources of classes in the deibert.zach package
          • school : Sources of classes in the deibert.zach.school package
            • billstolaws : Sources of classes in the deibert.zach.school.billstolaws package
              • tiles : Sources of classes in the deibert.zach.school.billstolaws.tiles package
              • uim : Sources of classes in the deibert.zach.school.billstolaws.uim package

Important files:

  • /dist/BillsToLaws.jar : The finished program
  • /bugs.ods : The list of bugs found
  • /README.html : This file
  • /board.html : Game Simulator (You can generate the chances)

Running the game
  1. Extract the archive (if you haven't already)
  2. Open the directory you extracted the archive into
  3. Open the BillsToLaws directory
  4. Open the dist directory
  5. Open the BillsToLaws.jar file

- or - 

    $ test -d BillsToLaws || unzip BillsToLaws.zip
    $ cd BillsToLaws/dist/
    $ java -jar BillsToLaws.jar

- or - 

    $ test -d BillsToLaws || unzip BillsToLaws.zip
    $ cd BillsToLaws/build/classes/
    $ java -cp . deibert.zach.school.billstolaws.Main
